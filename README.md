## Talvori

Talvori is a mobile language learning application built with Flutter that implements a **transparent spaced-repetition system (SRS)**.  
The project explores adaptive learning algorithms and user-controlled learning flows.

Key features:

• Transparent SRS learning system  
• Adaptive learning modes  
• Interactive learning interface  
• Supabase backend integration  
• Focus on user control and learning transparency  

Technologies:

Flutter  
Dart  
Supabase  
PostgreSQL  

Repository
